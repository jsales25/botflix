# BotFlix - Seu assistente pessoal para encontrar o filme perfeito

O BotFlix é uma aplicação web que te ajuda a escolher o filme ideal para qualquer ocasião. Basta descrever o que você quer assistir ou como está se sentindo, e o nosso assistente robô encontrará a recomendação perfeita para você.

## Funcionalidades

- **Recomendações personalizadas:** Receba sugestões de filmes com base no seu humor, companhia ou gênero preferido.
- **Interface simples e intuitiva:** Digite sua preferência e receba a sugestão instantaneamente.
- **Visualização de detalhes:** Veja o pôster, título, sinopse e avaliação de cada filme recomendado.

## Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **N8N:** Automação de fluxo de trabalho para buscar as recomendações.
- **The Movie Database (TMDB) API:** Fonte dos dados dos filmes.

## Como Executar

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/botflix.git
   ```
2. Abra o arquivo `index.html` no seu navegador de preferência.
